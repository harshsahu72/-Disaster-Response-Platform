# 🚨 AAPAT — Disaster Response Platform

**AAPAT (Aapad Aur Pratyutthan Tantra)** is a real-time disaster response and coordination platform designed to monitor, manage, and respond to emergencies across India.

It provides a **live map interface**, **incident tracking**, **responder coordination**, and **AI-based insights** to help authorities act quickly and efficiently during disasters.

---

## 🌐 Features

### 🗺️ Live Map

* Interactive India map with real-time incident markers
* Color-coded severity levels (Critical, High, Medium, Low)
* Hover tooltips and click-based incident details
* Heatmap and safe zone visualization

### 📡 Incident Reporting

* Users can report disasters in real-time
* Categorization by type (Flood, Cyclone, Fire, etc.)
* Severity and resource needs selection

### ⚡ Coordination Panel

* Task management (Pending, In Progress, Resolved)
* Resource allocation tracking
* Real-time progress updates

### 👨‍🚒 Responder Management

* View available and deployed teams
* Quick deployment options
* ETA and location tracking

### 📊 Analytics Dashboard

* Incident trends visualization
* Disaster type breakdown
* Data-driven insights

### 🔔 Alerts System

* Real-time alerts and warnings
* Critical updates highlighted
* AI-powered recommendations

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Visualization:** SVG-based interactive map
* **Styling:** Custom CSS with modern UI design
* **Data Handling:** Static JavaScript objects (can be extended to APIs)
